# SEO Map

## Current Baseline

### Page
- URL intent: homepage for `https://dii.ltd/`
- Page count: one production page
- Domain: `dii.ltd` via `/CNAME`

### Existing Meta In `/index.html`
- `<title>`: `Dhruvi Infinity Inspiration Ltd. | Build, Launch & Scale with Connected SaaS`
- `meta[name="description"]`: present
- `meta[name="theme-color"]`: present
- Open Graph:
  - `og:title`
  - `og:description`
  - `og:type`
  - `og:url`
  - `og:image`

### Heading Structure
- One `h1` in hero
- Section-level `h2` headings for each homepage block
- Nested `h3` headings in cards, product entries, timeline rows, and contact cards

## Internal Linking
- Internal links are all on-page anchor jumps
- Product links go directly to external destinations rather than internal detail pages
- No deeper content architecture exists yet inside this repo

## SEO Gaps
- No canonical tag
- No Twitter card tags
- No schema markup
- No `robots.txt` or sitemap in the repo
- No dedicated content pages yet for long-tail discovery
- Some visible copy is placeholder/editor-facing, which weakens perceived quality and search snippet alignment

## Near-Term SEO Work
- Clarify homepage message so title, H1, and on-page copy align more tightly
- Add canonical and Twitter meta
- Add Organization or WebSite schema if the homepage messaging is stabilized
- Build a future content architecture around founder guidance, visa-path guidance, and operations software explanations
